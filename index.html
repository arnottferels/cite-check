Disclaimer: This tool currently supports IEEE and Vancouver citation styles (more may be added later). It helps you quickly check if a reference has a DOI and if its year is within your selected range. It will generate a summary with percentages and display the parsed references. Since it's still in alpha, it may not be 100% accurate. If you have feedback or feature requests, feel free to share them in the GitHub discussions or submit a PR!

I created this tool because some journals require recent publications based on DOI. Counting manually in tools like Zotero, Mendeley, or Word can be slow and tedious (or is there an automatic way? I’m honestly not sure if there's a feature to do that automatically). This tool aims to make that process faster and easier. Keep in mind it’s still in early development, so it might not be perfect yet.
